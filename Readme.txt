This is a Web Scrapper, which uses Beautiful Soup Modules to Scrape the Hacker News Webstie, upto 2 Pages.

The News with votes more than '100' are only displayed in the terminal.

Usage:-

    Just Run the file in the Terminal. The output will given in the Terminal, Including the Link, Title and Votes.


The titles which have the Highest Votes are Showned at the top in the Terminal.
